Java关键字static和final

发表于 2017-04-17 | 分类于 Java

static关键字

修饰成员变量

- static修饰的成员变量不属于对象的数据结构
- static变量是类的变量，通常可以通过类名来引用static成员
- 被static修饰的成员变量和类的信息一起存储在方法区
- 无论创建了多少对象，一个类的static成员变量只有"一份"
