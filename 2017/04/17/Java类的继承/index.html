Java类的继承

发表于 2017-04-17 | 分类于 Java

Java类的继承(一)

extends关键字

- 通过extends关键字实现类的继承
- 子类可以继承父类的成员变量及成员方法，同时也可以自己定义自己的成员变量和成员方法
- Java语言不支持多继承，一个类只能继承一个父类，但一个父类可以有多个子类

继承中构造方法

- 子类构造方法需要通过super关键字调用父类的构造方法，super关键字必须位于子类构造方法的第一行
- 若子类的构造方法中没有调用父类的构造方法，编译器会自动的加入对父类无参构造方法的调用
- 若父类没有无参的构造方法，会有编译错误

向上造型

- 父类的引用指向子类的对象
- 定义父类型的引用可以指向子类的对象
- 父类型的引用只能访问父类所定义的成员和方法
