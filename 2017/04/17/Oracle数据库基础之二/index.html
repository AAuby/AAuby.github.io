Oracle数据库基础

数据类型

NUMBER

- 数字类型
- NUMBER(P,S)：P表示数字的总位数，S表示小数位数

CHAR

- 固定长度的字符类型，最大2000字节
- CHAR(N):N表示字节长度

VARCHAR2

- 变长的字符类型，最大4000字节
- VARCHAR2(N)：N表示最大可占用字节长度

DATE

- 定义日期时间的类型，长度是7个字节
- DD-MON-RR

修改表

修改表名

- RENAME old_name TO new_name;

增加列

- 列只能增加在最后，不能插入
- ALTER TABLE table_name ADD(Field DataType);

删除列

- ALTER TABLE table_name DROP(Field);

修改列

- 修改仅对以后插入的数据有效，不影响当前数据
- 如果把长度由大改小，有可能不成功
- ALTER TABLE table_name MODIFY(Field DataType);
